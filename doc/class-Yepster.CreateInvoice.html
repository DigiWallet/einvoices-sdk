Class CreateInvoice

Create Yepster invoice

Send as many details for the invoice as you want. It will be created as Draft, you need to approve (complete) and send the invoice thereafter. Minimal needed is 'name' for the recipient. The API key can be found in Yepster in your 'Profile' (upper-right menu)

Find a complete list of details at CreateInvoice::addDetail()
